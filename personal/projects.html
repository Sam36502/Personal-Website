Unfinished Projects

Very often, I'll come up with a cool idea for a thing I'd like to make, but --- so far, at least --- in 99% of cases, I give them up before they reach any kind of presentable state. This is a list of all currently "Ongoing" projects, because I never officially declare any of them dead. Because of this, some of the projects are actually still being worked on, but I'd like to have them here so if I forget them, they're already in the list.

Note: I made this a few years after the phenomenon became obvious, so this list is not extensive.

(In no particular order)

    Electronics:

  • Archimedes: A four-bit TTL logic computer built from scratch using a custom CPU architecture
  • A 4x4 pixel screen with custom-design driver board
  • A wrist-electronic debuggin panel with switches for toggling leads.
  • A tiny game-system with hand-soldered ROM cartridges (With 3-D printed housing)
  • Retroboy: Raspberry Pi in a 3d-printed gameboy case running retro game emulators

    • Writing:

  • Self-copied and bound copy of the epic of Gilgamesh as translated into Toki Pona here
  • Formatted version of Tolstoy's "Lucerne" formatted with LaTeX, printed and bound
  • My own computer science reference book written by me to be printed with a typewriter (Explains useful CompSci theories to be implemented practically)
  • Toki Pona Story written entirely in Toki Pona to avoid issues that come about from translation and to add to the Toki Pona literature pool
  • Julian's Utopia
  • Twin-Cities story with Amin

    • Programming:

  • Theltos: A generic DB frontend to make quickly setting up a DB with an accessible frontend easy
  • Consoleweb: A standard web format for being displayed in a text console that could be hosted like any normal HTTP server.
  • EasyMenus: A Java library to display console text in formatted pages with plenty of helpful options; A bit contrived, but kinda useful
  • Kalah Analysis Tool: KAT: A console tool for analysing games of Mancala (Kalah) and providing the user with the optimal moves to win
  • SimpleTUI: A TUI library for Java that would handle all input validation for you
  • HistoryGenerator: A program to simulate little people doing things across millenia and then generate archaelogical finds for you to "discover" detailing their history
  • EventAPI: An API to host important events that a client could check to remind you of important dates like birthdays (Obviously just got replaced by a calendar, lol)
  • Turing: A framework for decyphering text by finding the most likely cypher and applying them recursively

    • Conlanging/Worldbuilding:

  • Sugandese II: Electric Boogaloo: A conlang made for my friends and me that died after about 20 words
  • Khmyria: Made during school for no express purpose, but was fun to make
  • "Worldbuilding Project": lol yeah, that's about as far as it actually got. I started making the solar system it would be in, but didn't continue it after that.
  • Toki Ike: Weird/Evil sounding Toki Pona Relex for making fun "demonic rituals"
  • Sitelen Pakala: An alphabet (and font) for Toki Pona with the Germanic "Fraktur" look
  • A program to interpret The Malbolge Conlang phrases
  • A chronicle of everything that happened on the ClongCraft Minecraft server: I was going to ask everyone what they remember and compile it.

    • Game Dev

  • Shipbattles: A console-based implementation of the popular board-game Battleship; I technically finished it, but then wanted to add network multiplayer which never got added.
  • Snake/Tetris: On multiple occasions, I've wanted to implement Tetris or Snake, but never finished them.
  • Wilhelm I: Text based console game engine with MMO server
  • No-Name: A game where you are placed somewhere random in a world mirroring earth, and you have to use your knowledge of geography to find civilisation.
