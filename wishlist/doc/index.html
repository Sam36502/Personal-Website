delete /item/{item_id}
Deletes an item (itemItemIdDelete)
Takes an item's ID and permanently deletes that item from the database.

Path parameters

item_id (required)
Path Parameter — The item's ID

Responses

200

Successfully deleted the item

400

The provided Item ID was invalid
Up 
put /item/{item_id}
Updates an item's details (itemItemIdPut)
Takes an item's ID and item object and updates the provided elements to whatever was provided. Every element provided is optional, but if it is provided it will overwrite that element in the db. This is the same for the links, but you must provide all the links you want to keep; if a link is omitted but the link attribute provided, it will be deleted.

Path parameters

item_id (required)
Path Parameter — The item's ID

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body Item (required)
Body Parameter — The item to overwrite with (all elements are optional)

Responses

200

Successfully updated the item

400

The provided Item ID or item object was invalid
Up

get /item/{item_id}
Gets a single item with its ID (itemItemIdGet)
Takes the item's ID and retrieves that item's information alone

Path parameters

item_id (required)
Path Parameter — The item's ID

Return type

Item

Example data

Content-Type: application/json
{
  "item_id" : 432,
  "price" : 7178,
  "name" : "The Hobbit Book",
  "description" : "The deluxe edition of the Hobbit",
  "links" : [ {
    "hyperlink" : "https://www.bookdepository.com/Hobbit-J-R-R-Tolkien/9780007118359?ref=pd_gw_1_pd_gateway_1_1",
    "text" : "Book Depository"
  }, {
    "hyperlink" : "https://www.bookdepository.com/Hobbit-J-R-R-Tolkien/9780007118359?ref=pd_gw_1_pd_gateway_1_1",
    "text" : "Book Depository"
  } ],
  "status" : {
    "status_id" : 0,
    "name" : "Available",
    "description" : "The items which are avaliable to buy"
  }
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Successfully retrieved the item Item

400

The provided Item ID was invalid

post /token
Creates access token (tokenPost)
Takes basic auth values from the user and returns an access token which can be used to access the private resources they have access to until the token expires.

Request headers

Return type

Token

Example data

Content-Type: application/json
{
  "access_token" : "ZDkxN2Q0M2M0ODRkNDM4NDdkN2MzMmE2MTMxNjQ5N2E=",
  "token_type" : "Bearer",
  "expires_in" : 7200
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Successfully created access token Token

401

Incorrect credentials provided
